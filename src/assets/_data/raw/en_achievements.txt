##achievement.mode.arena : arena
##achievement.mode.inf_arena : infinite arena
##achievement.mode.reverse : reverse
##achievement.mode.unknown : ???

##achievement.grade.none : not achived yet
##achievement.grade.normal : normal grade
##achievement.grade.elite : elite grade

##achievement.trample : Stompy
##achievement.trample.description
Crush at least 1 enemy ground robot by landing a colony on it.

##achievement.impossible : Impossibly Difficult
##achievement.impossible.description
Win while having a difficulty set to more than 200%.

##achievement.speedrunning : Speedrunning
##achievement.speedrunning.description
Achieve a victory under 15 minutes.

##achievement.t3engineer : Gotta Catch 'Em All
##achievement.t3engineer.description
Discover all tier 3 drones.

##achievement.t3less : Tech Limitations
##achievement.t3less.description
Achieve a victory without creating a single tier 3 drone.

##achievement.cheapbuild10 : Small Deck
##achievement.cheapbuild10.description
Win while having a drone build that uses no more than 10 points allocated.

##achievement.hightension : High Tension
##achievement.hightension.description
Win on the smallest map size possible without destroying any of the enemy bases.
The "creep bases" difficulty option should not be 0.

##achievement.solobase : Solo Base Run
##achievement.solobase.description
Win without building any new colonies.

##achievement.uiless : Minimal UI
##achievement.uiless.description
Win while having the extra graphical user interface turned off.
Toggle the "user interface mode" in "extra" tab into "minimal".

##achievement.powerof3 : A Power of Three
##achievement.powerof3.description
Win a battle without using one of the four factions.
In other words, use only three factions during the battle.

##achievement.victorydrag : Dragged Victory
##achievement.victorydrag.description
Win a battle that was going for no less than 2 hours.

##achievement.tinyradius : Sealed Radius
##achievement.tinyradius.description
Clear a level without ever increasing the influence radius.

##achievement.t1army : T1 Army
##achievement.t1army.description
Clear the level while having only tier 1 combat units.
In other words, do not create any tier 2 or 3 combat units during the game to get this achievement.

##achievement.groundwin : Ground Salvo
##achievement.groundwin.description
Destroy the dreadnought in the moment when it's close to the ground.

##achievement.turretdamage : Turret Barrage
##achievement.turretdamage.description
Deal at least 25% damage to the dreadnought using ground turrets.

##achievement.leet : 1337
##achievement.leet.description
Win a match with a seed set to 1337. This seed enables some special rules, so be prepared.

##achievement.antidominator : Anti-domination
##achievement.antidominator.description
Destroy all dominator class enemies before the final wave countdown reaches zero.

##achievement.infinite : Infinite Battle
##achievement.infinite.description
Survive for at least 35 waves in infinite arena mode.

##achievement.nopeeking : No Peeking
##achievement.nopeeking.description
Win a game without looking at the evolution tab even once.
In other words, don't press ALT (on keyboard) or R2 (on gamepad) during the entire match.

##achievement.colonyhunter : Colony Hunter
##achievement.colonyhunter.description
Destroy at least 3 colonies during the victorious reverse match.

##achievement.groundcontrol : Ground Control
##achievement.groundcontrol.description
During any moment of the victorious reverse match, have at least 20 rocket turrets exist on the battlefield.

##achievement.atomicfinisher : A-bomb Launch Detected
##achievement.atomicfinisher.description
Win a game by launching an atomic bomb. An atomic bomb is available at 150% tech progress level.

##achievement.nonstop : Nonstop
##achievement.nonstop.description
Get to a victory in any mode without using a pause feature.
Using the exit button to activate a pause counts as pausing too.

##achievement.secret : The True Ending
##achievement.secret.description
This game has a true, canonical ending.
It's not simple to find, but it's there!

##achievement.terminal : Terminator
##achievement.terminal.description
This is probably the most obscure achievement this game has.
The hint is there, for what it's worth.

##achievement.spectator : Spectator
##achievement.spectator.description
Watch a splash screen simulation until the end. Enjoy the cinematic camera while at it.

##achievement.gladiator : Gladiator
##achievement.gladiator.description
Submit your results and enter the global leadership table!
